Drawn Together

Drawn Together by Girls Make Games

Created by two Teen campers at Girls Make Games. As their counselor, I mentored them and taught them how to use Unity using GMG materials, and guided them through the game design process. We used Ink for the visual novel segments, and I wrote all scripts outside of basic platformer scripts provided by GMG. These girls were so fun to work with, and so talented to be able to make a complete game in only 2.5 weeks!

Game instructions: Click on text boxes to advance dialogue. Use WASD and arrow keys to move. The pencil must hit all pencil sharpener icons before the pen is able to begin, and the level is complete when the pen has hit all vertices.
